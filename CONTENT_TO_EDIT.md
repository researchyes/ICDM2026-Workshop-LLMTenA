# Quick content checklist

Search for the following placeholders and replace them:

- `{{WORKSHOP TITLE}}`
- `{{FULL WORKSHOP TITLE}}`
- `{{WORKSHOP SHORT NAME}}`
- `{{WORKSHOP DATE}}`
- `{{CITY / FORMAT}}`
- `{{SUBMISSION DEADLINE}}`
- `{{CONTACT EMAIL}}`
- `{{SUBMISSION URL}}`
- `{{TIMEZONE}}`
- `{{Organizer 1}}`, `{{Organizer 2}}`, etc.
- `{{Speaker Name}}`, `{{Affiliation}}`
- `{{DATE}}` placeholders across pages

Recommended order:

1. Edit `index.html`
2. Edit `cfp.html`
3. Edit `submission.html`
4. Edit `organizers.html`
5. Edit `schedule.html`
