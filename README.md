# ICDM2026-Workshop-LLMTenA starter website

This folder contains a GitHub Pages–ready static website for your ICDM 2026 workshop repository.

## Files

- `index.html` — homepage
- `cfp.html` — call for papers
- `submission.html` — submission instructions
- `schedule.html` — program / schedule
- `organizers.html` — organizers and program committee
- `style.css` — shared styles
- `assets/banner.svg` — placeholder illustration
- `CONTENT_TO_EDIT.md` — quick checklist of placeholders to replace

## How to publish on GitHub Pages

1. Open your repository `ICDM2026-Workshop-LLMTenA`.
2. Upload all files from this folder to the repository root.
3. Commit the files to the `main` branch.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
6. Save.
7. After GitHub finishes publishing, your site should be available at:

```text
https://YOUR-USERNAME.github.io/ICDM2026-Workshop-LLMTenA/
```

## Notes

- Keep the relative links as they are. They are already set up for a project site.
- Replace placeholders in double braces such as `{{WORKSHOP TITLE}}`.
- You can remove pages you do not need, but if you remove one, also delete its link from the navigation menu.
- If you later want a custom domain, add it in **Settings → Pages**.
